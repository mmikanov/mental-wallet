# Product Overview

Mental Health Wallet is a mobile-first app that unifies scattered mental health coping tools into a personalized, habit-forming toolkit. Users collect "cards" (coping tools, exercises, reminders, trackers) from a curated library or create their own using a generic control system.

## Core Concepts

- **Wallet**: The main interface using a stacked card layout (inspired by Apple Wallet)
- **Cards**: Individual coping tools with a universal shell (title, description, icon, background) and configurable controls
- **Controls**: Reusable UI elements (text inputs, mood sliders, choice buttons, etc.) that compose custom tools
- **Completions**: Logged usage of a card, capturing all control values and timestamps
- **Origin Badges**: Cards are tagged as "library" (curated, read-only), "community" (shared, read-only), or "my_tool" (user-created, editable)

## Key Behaviors

- Cards can be focused, expanded in-place, reordered, archived, and restored
- Each card tracks usage count, streaks, and mood correlation
- Reminders can be set per-card with configurable frequency
- Users can duplicate read-only cards to make editable copies
- The MVP includes 18–21 hand-curated cards across categories: Grounding, Cognitive Reframing, Body & Sensory, Daily Check-In, Self-Compassion, and Connection

## Target Users

- **Primary (MVP)**: Health-conscious consumers ages 18–55 interested in mental wellness and self-improvement
- **Secondary (Post-MVP)**: Licensed therapists assigning tools to clients
